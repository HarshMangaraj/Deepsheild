# DeepShield Backend — Python Dependencies
#
# WHY EACH PACKAGE:
#
# fastapi           — The web framework. Handles routing, validation, docs.
# uvicorn           — The server that actually runs FastAPI. Like Next.js's
#                     dev server but for Python. `uvicorn main:app`
# python-multipart  — Required for FastAPI to parse file uploads (multipart/form-data).
#                     Without this, `UploadFile` won't work.
# torch             — PyTorch, the deep learning library. Runs the model.
# torchvision       — Adds image transforms + pretrained model utilities to torch.
# timm              — "PyTorch Image Models" — huge library of pretrained vision models.
#                     Lets us do `timm.create_model("efficientnet_b4", pretrained=True)`
# Pillow            — PIL fork. Opens/converts images (JPEG, PNG, WEBP, etc).
# huggingface_hub   — Downloads pretrained model weights from HuggingFace.
# python-dotenv     — Loads .env files for config (API keys, settings).
